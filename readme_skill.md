# Claude Code 使用技巧

## 快速开始

Claude Code 是 Anthropic 官方的 CLI 工具，帮助你完成软件工程任务。

### 基本使用
```bash
# 启动 Claude Code
claude

# 直接执行任务
claude "修复 main.py 中的 bug"
```

## 常用命令

### 项目初始化
- `/init` - 初始化项目文档
- `/help` - 查看帮助信息

### 任务管理
- `/clear` - 清空会话
- `/tasks` - 查看后台任务
- `/loop` - 循环执行任务

### 配置
- `/fast` - 切换快速模式
- `/context` - 查看当前上下文

## 工作流程

### 1. 探索代码库
```
"帮我探索这个项目的结构"
"找出所有的 Python 文件"
```

### 2. 制定计划
使用 `/plan` 命令或让 Claude 自动进入计划模式

### 3. 执行任务
- 编辑文件
- 运行测试
- 重构代码

### 4. 验证结果
- 运行测试
- 检查代码
- 提交更改

## 最佳实践

### 清晰的指令
- 具体描述你想要做什么
- 提供上下文和相关文件路径
- 说明预期结果

### 分步骤执行
- 将大任务拆分为小步骤
- 每完成一步验证结果
- 使用 TodoWrite 跟踪进度

### 利用上下文
- Claude 会记住之前的对话
- 可以引用之前的讨论
- 逐步完善需求

## 快捷键

- `Ctrl+C` - 中断当前操作
- `Ctrl+D` - 退出会话
- `↑/↓` - 浏览历史命令

## 常见问题

### Q: Claude 不理解我的需求？
A: 尝试更具体地描述，提供文件路径和代码示例。

### Q: 如何让 Claude 只探索不修改？
A: 使用 "探索"、"分析"、"查看" 等词汇，明确表示不需要修改。

### Q: 任务执行失败怎么办？
A: 检查错误信息，重新描述需求，或拆分为更小的步骤。
