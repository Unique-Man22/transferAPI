# DouBao Transfer API - 项目概述

## 项目简介
DouBao Transfer API 是一个 Flask 中转服务，用于将 Anthropic Claude API 请求格式转换为 OpenAI 兼容格式，主要用于连接火山方舟（Volcengine）的豆包大模型。

## 核心价值
- **API兼容性转换**：让支持 Anthropic Claude API 的客户端能够无缝使用火山方舟的豆包模型
- **工具调用支持**：完整支持 Function calling（工具调用）功能
- **Token统计监控**：实时监控中转前后的 Token 使用量和损耗

## 应用场景
1. **客户端兼容**：已有 Anthropic Claude API 客户端，希望接入豆包模型
2. **成本监控**：需要精确统计 API 调用成本和格式转换损耗
3. **服务中转**：在企业内部搭建统一的 AI 服务网关

## 技术特点
- **轻量级设计**：单文件 Flask 应用，易于部署和维护
- **模块化设计**：核心转换逻辑与 Token 统计分离
- **可扩展性**：支持 CORS、速率限制等可选功能
- **生产就绪**：完善的错误处理和日志记录
