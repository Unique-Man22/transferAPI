# DouBao Transfer API - 代码架构分析

## 整体架构概述

这是一个单文件 Flask 应用，采用 **请求-响应转换模式**：

```
客户端 (Anthropic格式)
    ↓
Flask 应用 (main.py)
    ↓ 格式转换: Anthropic → OpenAI
火山方舟 API (豆包模型)
    ↓
Flask 应用 (main.py)
    ↓ 格式转换: OpenAI → Anthropic
客户端 (Anthropic格式)
```

### 5. 核心工具函数

```python
def extract_text_from_content(content):
    """从Anthropic格式的content列表中提取纯文本"""
    if isinstance(content, str):
        return content
    elif isinstance(content, list):
        text_parts = []
        for item in content:
            if isinstance(item, dict):
                if item.get("type") == "text":
                    text_parts.append(item.get("text", ""))
                elif item.get("type") == "tool_result":
                    text_parts.append(extract_text_from_content(item.get("content", "")))
        return "".join(text_parts)
    return ""
```
**功能**：
- 支持字符串直接返回
- 支持文本块提取
- 支持工具结果块递归提取

#### `convert_system_prompt(system)` (第 152-163 行)

```python
def convert_system_prompt(system):
    """将system prompt转换为字符串格式"""
    if isinstance(system, str):
        return system
    elif isinstance(system, list):
        # 假设是文本块列表
        return "".join([item.get("text", "") for item in system if isinstance(item, dict)])
```

**消息历史转换**：
```python
def convert_messages_to_openai(messages, system):
    """将Anthropic消息历史转换为OpenAI格式"""
    openai_messages = []
    
    # 添加系统消息
    if system:
        openai_messages.append({"role": "system", "content": system})
    
    # 转换消息历史
    for msg in messages:
        role = msg.get("role")
        content = msg.get("content", "")
        
        if role == "assistant" and isinstance(content, list):
            # 处理tool_use消息
            for item in content:
                if isinstance(item, dict) and item.get("type") == "tool_use":
                    openai_messages.append({
                        "role": "assistant",
                        "content": None,
                        "tool_calls": [{
                            "id": item.get("id", ""),
                            "type": "function",
                            "function": {
                                "name": item.get("name", ""),
                                "arguments": json.dumps(item.get("input", {}))
                            }
                        }]
                    })
        elif role == "user" and isinstance(content, list):
            # 处理tool_result消息
            for item in content:
                if isinstance(item, dict) and item.get("type") == "tool_result":
                    openai_messages.append({
                        "role": "tool",
                        "tool_call_id": item.get("tool_use_id", ""),
                        "content": extract_text_from_content(item.get("content", ""))
                    })
        else:
            # 普通文本消息
            openai_messages.append({
                "role": role,
                "content": extract_text_from_content(content)
            })
    
    return openai_messages
```

```python
def convert_openai_to_claude(openai_response):
    """将OpenAI格式响应转换为Claude格式"""
    choice = openai_response.get("choices", [{}])[0]
    message = choice.get("message", {})
    
    claude_response = {
        "id": f"msg_{uuid.uuid4().hex[:16]}",
        "type": "message",
        "role": "assistant",
        "content": [],
        "model": VOLC_MODEL,
        "stop_reason": choice.get("finish_reason", "stop"),
        "usage": {
            "input_tokens": openai_response.get("usage", {}).get("prompt_tokens", 0),
            "output_tokens": openai_response.get("usage", {}).get("completion_tokens", 0)
        }
    }
    
    # 处理工具调用
    if message.get("tool_calls"):
        for tool_call in message.get("tool_calls", []):
            claude_response["content"].append({
                "type": "tool_use",
                "id": tool_call.get("id", ""),
                "name": tool_call.get("function", {}).get("name", ""),
                "input": json.loads(tool_call.get("function", {}).get("arguments", "{}"))
            })
    else:
        # 普通文本响应
        claude_response["content"].append({
            "type": "text",
            "text": message.get("content", "")
        })
    
    return claude_response
```

```python
    """处理流式响应生成器"""
    for line in stream_response.iter_lines():
        if line:
            line = line.decode('utf-8')
            if line.startswith('data: '):
                data = line[6:]
                if data == '[DONE]':
                    break
                try:
                    chunk = json.loads(data)
                    claude_chunk = convert_openai_chunk_to_claude(chunk)
                    yield f"data: {json.dumps(claude_chunk)}\n\n"
                except json.JSONDecodeError:
```

## token_stats.py 架构分析

### 类结构
```python
class TokenStatistics:
    """Token统计类"""
    
    def __init__(self):
        self.claude_tokenizer = None
        self.doubao_tokenizer = None
        self._init_tokenizers()
        
        # 累积统计
        self.accumulated_stats = {
            "total_requests": 0,
            "claude": {"input_tokens": 0, "output_tokens": 0, "total_tokens": 0},
            "doubao": {"input_tokens": 0, "output_tokens": 0, "total_tokens": 0},
            "loss": {"input_loss": 0, "output_loss": 0, "total_loss": 0}
        }
```

### 核心方法
1. **`_init_tokenizers()`**：初始化分词器
2. **`count_tokens(text, tokenizer_type)`**：统计 Token 数量
3. **`record_request()`**：记录单次请求统计
4. **`get_accumulated_stats()`**：获取累积统计
5. **`reset_stats()`**：重置统计数据

### 设计模式
- **单例模式**：通过 `get_token_stats()` 函数获取全局实例
- **策略模式**：支持 tiktoken 和字符估算两种分词策略
- **观察者模式**：定时任务观察统计变化

## 架构设计特点

### 1. 单一职责原则
- `main.py`：HTTP 服务和格式转换
- `token_stats.py`：Token 统计功能
- 每个函数职责明确，不超过 50 行

### 2. 开闭原则
- 可选依赖通过条件导入实现
- 配置通过环境变量扩展
- 新功能可通过添加模块实现

### 3. 依赖倒置原则
- 高层模块不依赖低层模块细节
- 通过接口（函数签名）定义依赖
- Token 统计模块独立于主应用

### 4. 接口隔离原则
- 每个 API 端点功能单一
- 工具函数职责明确
- 配置项分类清晰

### 5. 迪米特法则
- 模块间通过明确定义的接口通信
- 内部实现细节封装良好
- 全局状态通过单例管理
