# DouBao Transfer API - 技术栈分析

## 核心编程语言
- **Python 3.x**：主要开发语言，版本要求未明确指定但建议 3.7+

## Web 框架
- **Flask**：轻量级 Web 框架，用于构建 RESTful API 服务
  - 版本：未指定，最新稳定版即可
  - 用途：HTTP 请求处理、路由、中间件

## HTTP 客户端
- **requests**：HTTP 请求库
  - 用途：转发请求到火山方舟 API
  - 特性：支持超时设置、错误处理

## 配置管理
- **python-dotenv**：环境变量管理
  - 用途：从 .env 文件加载配置
  - 配置项：API 密钥、服务端口、模型选择等

## 可选依赖（功能增强）

### CORS 支持
- **flask-cors**：跨域资源共享支持
  - 用途：允许跨域 API 调用
  - 特性：安装后自动启用

### 速率限制
- **flask-limiter**：API 速率限制
  - 用途：防止 API 滥用
  - 配置：默认 "100 per minute"

### Token 统计
- **tiktoken>=0.5.0**：OpenAI 分词器
  - 用途：精确统计 Claude 和豆包模型的 Token 使用量
  - 特性：使用 cl100k_base 编码器
  - 降级方案：未安装时使用字符估算

### 定时任务
- **apscheduler**：高级 Python 调度器
  - 用途：定时记录 Token 统计信息
  - 配置：每分钟自动记录一次

## 开发工具链

### 包管理
- **pip**：Python 包管理器
- **requirements.txt**：依赖声明文件

### 版本控制
- **Git**：代码版本管理
- **.gitignore**：忽略文件配置

### 文档

## 技术架构特点

### 1. 条件导入机制
```python
try:
    from flask_cors import CORS
    CORS_AVAILABLE = True
except ImportError:
    CORS_AVAILABLE = False
```
- 支持可选依赖的优雅降级
- 增强功能可选择性安装

### 2. 模块化设计
- **main.py**：主应用程序（300+行）
- **token_stats.py**：Token 统计模块
- **test_token_stats.py**：测试文件

### 3. 配置驱动
- 环境变量优先
- .env 文件支持
- 代码中默认值作为后备

## 技术选型优势

### Flask 的优势
1. **轻量级**：启动快，资源占用少
2. **灵活性**：易于扩展和定制
3. **成熟生态**：丰富的扩展库
4. **学习曲线低**：适合快速开发和维护

### tiktoken 的优势
1. **准确性**：与 OpenAI 官方分词器一致
2. **性能**：本地分词，无需网络请求
3. **兼容性**：支持 Claude 和豆包模型的统一分词

### apscheduler 的优势
1. **可靠性**：支持持久化存储
2. **灵活性**：多种调度策略
3. **易用性**：简单的 API 设计
