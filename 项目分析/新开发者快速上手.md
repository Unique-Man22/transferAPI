## 项目核心概念

### 1. 什么是 DouBao Transfer API？
这是一个 **API 格式转换服务**，让支持 Anthropic Claude API 的客户端能够无缝使用火山方舟的豆包大模型。

**简单比喻**：就像是一个翻译官，把 Claude 的语言翻译成豆包能听懂的语言。

### 2. 核心转换流程
```
你的应用 (Claude格式请求)
    → 本服务 (格式转换)
    → 火山方舟豆包模型
    → 本服务 (格式转换)
    → 你的应用 (Claude格式响应)
```

### 3. 主要功能特性
- ✅ **格式转换**：Claude ↔ 豆包双向转换
- ✅ **工具调用**：完整支持 Function calling
- ✅ **流式响应**：支持实时流式输出
- ✅ **Token统计**：监控使用量和损耗
- ✅ **健康检查**：服务状态监控
- ✅ **认证保护**：API密钥认证

## 代码阅读指南

### 从哪里开始读代码？

#### 第一步：看主程序入口
打开 `main.py`，直接跳到最底部：
```python

    setup_logging()      # 设置日志
    setup_scheduler()    # 启动定时任务
    app.run(...)         # 启动Flask服务
```

#### 第二步：看API端点
在 `main.py` 中搜索 `@app.route`：
- `/v1/messages` - 主要API端点
- `/health` - 健康检查
- `/stats` - Token统计

#### 第三步：看核心转换函数
搜索这些关键函数：
- `extract_text_from_content()` - 内容提取
- `convert_system_prompt()` - 系统提示转换
- `validate_message()` - 消息验证

#### 第四步：看Token统计模块
打开 `token_stats.py`，看 `TokenStatistics` 类：
- `record_request()` - 记录请求统计
- `count_tokens()` - 统计Token数量
- `get_accumulated_stats()` - 获取统计信息
